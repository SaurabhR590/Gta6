A visually immersive **React + GSAP** animated landing page inspired by the *Grand Theft Auto* aesthetic. This project demonstrates advanced animation techniques including SVG masking, cinematic transitions, and mouse-based parallax effects.

---

## 🚀 Features

* 🎭 **SVG Mask Intro Animation**

  * Animated "VI" mask reveal using GSAP timeline
  * Smooth transition into main content

* 🎬 **Cinematic UI**

  * Multi-layered visuals (sky, background, character)
  * Bold GTA-inspired typography

* 🌀 **Advanced GSAP Animations**

  * Timeline sequencing
  * Scale, rotation, opacity transitions

* 🖱️ **Mouse Parallax Effect**

  * Real-time cursor-based movement
  * Creates depth illusion

* 📱 **Modern Layout**

  * Clean navbar
  * Structured and scalable UI

## 🎯 How It Works

### 🔹 Intro Animation

* SVG mask expands and fades using GSAP timeline
* Once animation completes → main content loads

### 🔹 Content Animation

* Elements animate into place:

  * `.main`
  * `.sky`
  * `.bg`
  * `.character`
  * `.text`

### 🔹 Parallax Interaction

* Mouse movement dynamically shifts layers
* Creates smooth depth illusion
